# Contribution Guidelines

Thank you for considering contributing to The Data Career Playbook!

To ensure this list remains high-quality, please adhere to the following guidelines.

## How to Contribute

1.  **Check for Duplicates**: Search the list to ensure the resource hasn't already been added.
2.  **Quality Check**: Only add resources that are widely used, well-documented, or highly unique. Avoid self-promotion unless the tool is genuinely "awesome" and free/open-source.
3.  **Category**: Add your link to the most appropriate section (e.g., Python Libraries, SQL Tools).

## Formatting

Please follow this format for all entries:

`- [Resource Name](URL) - A short, objective description of the resource.`

### Example:

* **Good:** `[Pandas](https://pandas.pydata.org/) - Fast, powerful, flexible and easy to use open source data analysis and manipulation tool.`
* **Bad:** `[Pandas](https://pandas.pydata.org/) - The best tool ever!!!` (Avoid subjective claims like "best").

## Submitting

1.  Fork this repository.
2.  Create a new branch (`git checkout -b add-new-resource`).
3.  Commit your changes.
4.  Push to the branch.
5.  Open a Pull Request (PR) and describe what you added.
