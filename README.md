## Platforms & BI Tools
*Tools for visualizing and reporting on data.*

- [Tableau](https://www.tableau.com/) - A visual analytics platform transforming the way we use data to solve problems.
- [Power BI](https://powerbi.microsoft.com/) - Interactive data visualization BI tool from Microsoft.
- [Looker Studio](https://lookerstudio.google.com/) - Free tool to turn your data into informative, easy to read, easy to share, and fully customizable dashboards.
- [Metabase](https://www.metabase.com/) - The easiest way for everyone in your company to ask questions and learn from data.

## Python Libraries
*Libraries for data manipulation and analysis.*

- [Pandas](https://pandas.pydata.org/) - Fast, powerful, flexible and easy to use open source data analysis and manipulation tool.
- [NumPy](https://numpy.org/) - The fundamental package for scientific computing with Python.
- [Matplotlib](https://matplotlib.org/) - Comprehensive library for creating static, animated, and interactive visualizations.
- [Seaborn](https://seaborn.pydata.org/) - Statistical data visualization based on matplotlib.

## R Packages
*Essential packages for analytics in R.*

- [dplyr](https://dplyr.tidyverse.org/) - A grammar of data manipulation.
- [ggplot2](https://ggplot2.tidyverse.org/) - A system for declaratively creating graphics.
- [shiny](https://shiny.rstudio.com/) - Easy web apps for data science.

## SQL Tools
*Tools for querying and managing databases.*

- [dbt (data build tool)](https://www.getdbt.com/) - Enables data analysts and engineers to transform their data using the same practices that software engineers use.
- [DBeaver](https://dbeaver.io/) - Free multi-platform database tool for developers, database administrators, and analysts.

## Public Datasets
*Where to find data for your projects.*

- [Kaggle Datasets](https://www.kaggle.com/datasets) - A vast repository of community-uploaded datasets.
- [Google Dataset Search](https://datasetsearch.research.google.com/) - A search engine for datasets.
- [Awesome Public Datasets](https://github.com/awesomedata/awesome-public-datasets) - A topic-centric list of HQ open datasets.

## Learning Resources
*Courses, books, and tutorials.*

- [Google Data Analytics Certificate](https://grow.google/certificates/data-analytics/) - Professional training designed by Google.
- [DataCamp](https://www.datacamp.com/) - Interactive learning for R, Python, and SQL.
